# .dotfiles

These are my personal dotfiles setup.

### Install dependencies

#### Mac
```
brew install neovim rbenv fzf
```

### Symlink files

```
cd
git clone git@github.com:lollkebear/.dotfiles.git
./install
```

### Set up local gitconfig
```
nvim ~/.gitconfig.local

[user]
  name = Your Name
  email = your@email.com
```
